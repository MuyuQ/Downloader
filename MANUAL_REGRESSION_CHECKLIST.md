# WYDownloader 手工回归清单

适用版本：2026-03-09 代码修复批次

## 0. 准备

1. 使用 Visual Studio 2019/2022 打开 `WYDownloader.sln`
2. 选择 `Debug|AnyCPU`，执行“生成解决方案”
3. 启动程序，确认主窗口正常显示

## 1. 下载与镜像切换

1. 在 `Resources/Config/config.ini` 配置一个可下载项
2. 若配置多镜像，故意让第一个地址失效
3. 点击“开始下载”
4. 预期：
   - 首地址失败后自动尝试下一地址
   - 进度与速度持续刷新
   - 下载完成后弹出成功提示

## 2. 断点续传分支

1. 在配置中确保 `EnableResume=true`
2. 下载大文件，中途点击“取消下载”
3. 再次点击“开始下载”同一文件
4. 预期：
   - 继续下载而非从 0 开始
   - 最终文件可正常使用

## 3. 关闭流程一致性

1. 下载进行中点击右上角关闭按钮
2. 在确认弹窗中先点“否”，再点关闭并选择“是”
3. 预期：
   - 仅在关闭流程中出现一次确认
   - 窗口关闭过程无二次确认
   - 不出现“下载失败”误报弹窗

## 4. ZIP 解压场景

1. 下载普通 ZIP，开启自动解压
2. 下载空 ZIP（无 entry），开启自动解压
3. 预期：
   - 普通 ZIP 正常解压，进度可见
   - 空 ZIP 不报错，直接完成

## 5. 取消与解压中断

1. 下载并进入自动解压阶段时点击“取消下载”
2. 预期：
   - UI 提示“解压已取消”或“操作已取消”
   - 应用保持可操作，不崩溃

## 6. 配置回退与日志

1. 临时重命名 `Resources/Config/config.ini` 后启动
2. 检查日志目录 `logs/`
3. 预期：
   - 应用仍可启动（走嵌入或默认配置）
   - 日志中可看到配置读取失败与回退信息
