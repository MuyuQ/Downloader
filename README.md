# WY直链下载器

一个功能强大的Windows桌面直链下载器，支持HTTP/HTTPS协议的文件下载和ZIP文件自动解压功能。

## 功能特点

- ✅ **直链下载**: 支持HTTP/HTTPS协议的文件下载
- ✅ **自动解压**: 下载完成后可自动解压ZIP文件
- ✅ **实时进度**: 显示下载进度、速度和文件大小
- ✅ **美观界面**: 现代化的WPF界面设计，支持背景图片
- ✅ **拖拽支持**: 支持拖拽URL链接到程序窗口
- ✅ **智能路径**: 默认保存到程序所在目录，支持自定义路径
- ✅ **文件管理**: 自动处理重名文件，避免覆盖
- ✅ **兼容性强**: 基于.NET Framework 4.8，兼容Windows 7及以上系统

## 系统要求

- Windows 7 SP1 或更高版本
- .NET Framework 4.8 或更高版本
- 至少 50MB 可用磁盘空间

## 使用方法

1. **启动程序**: 双击 `WYDownloader.exe` 启动程序
2. **输入链接**: 在"下载链接"输入框中输入要下载的文件URL
   - 支持直接粘贴链接
   - 支持拖拽链接到程序窗口
   - 按回车键快速开始下载
3. **选择路径**: 点击"浏览..."按钮选择文件保存位置（默认为程序所在目录）
4. **设置选项**: 勾选"自动解压ZIP文件"可在下载完成后自动解压ZIP文件
5. **开始下载**: 点击"开始下载"按钮开始下载
6. **监控进度**: 实时查看下载进度、速度和状态信息

## 界面说明

- **下载链接**: 输入要下载的文件URL地址
- **保存路径**: 选择文件保存的目录位置
- **自动解压ZIP文件**: 勾选后会在ZIP文件下载完成后自动解压
- **进度条**: 显示当前下载进度百分比
- **状态信息**: 显示当前操作状态和文件信息
- **速度显示**: 实时显示下载速度

## 支持的文件类型

- 所有通过HTTP/HTTPS协议可访问的文件
- 特别优化了ZIP文件的处理，支持自动解压功能
- 自动识别文件名，智能处理重名文件

## 注意事项

- 确保网络连接正常
- 确保有足够的磁盘空间存储下载文件
- 某些网站可能有防盗链保护，可能无法直接下载
- 下载大文件时请保持程序运行，避免中途关闭
- ZIP解压功能需要文件完整下载后才能执行

## 技术特性

- 基于 WPF (Windows Presentation Foundation) 开发
- 使用 HttpClient 进行网络请求
- 支持异步下载，不阻塞用户界面
- 内置进度回调和速度计算
- 使用 System.IO.Compression 进行ZIP解压
- 支持取消下载操作
- 完善的错误处理机制
