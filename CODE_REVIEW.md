# WYDownloader 项目代码审查报告

## 项目概述

WYDownloader 是一个基于 WPF 的直链下载器应用程序，主要功能包括：
- HTTP/HTTPS 直链下载
- 断点续传支持
- ZIP 文件自动解压
- 下载暂停/恢复功能
- 配置化管理下载链接

## 模块分析

### 1. App.xaml.cs - 应用程序入口

**当前状态：** 几乎空白，仅有默认代码

**发现的问题：**
- 缺少全局异常处理机制
- 未记录应用程序启动日志
- 无启动参数处理

**改进建议：**
- 添加全局异常捕获（DispatcherUnhandledException、TaskScheduler.UnobservedTaskException）
- 添加应用程序启动/关闭日志
- 考虑单实例运行检测

### 2. MainWindow.xaml.cs - 主窗口
