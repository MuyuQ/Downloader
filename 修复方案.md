## 🔧 详细修复方案

### 修复 1：EnableResume 配置生效

**文件：** `MainWindow.xaml.cs`  
**位置：** 约 464-550 行（DownloadFileAsync 相关逻辑）

**当前问题：**
```csharp
// 读取了配置但未使用
bool enableResume = configManager.GetEnableResume();
long existingBytes = 0;
if (File.Exists(partPath))
{
    existingBytes = new FileInfo(partPath).Length;  // 始终续传
}
```

**修复代码：**
```csharp
bool enableResume = configManager.GetEnableResume();
long existingBytes = 0;

// 根据配置决定是否续传
if (enableResume && File.Exists(partPath))
{
    existingBytes = new FileInfo(partPath).Length;
    Debug.WriteLine($"[LOG] 启用续传：{existingBytes} bytes");
}
else if (!enableResume && File.Exists(partPath))
{
    // 禁用续传时删除旧的 part 文件
    File.Delete(partPath);
    Debug.WriteLine("[LOG] 禁用续传，已删除旧 part 文件");
}

// 在发送 Range 请求前也要检查
if (enableResume && existingBytes > 0)
{
    request.Headers.Range = new RangeHeaderValue(existingBytes, null);
}
```

**测试验证：**
1. 设置 `EnableResume=false`，下载文件，观察是否从 0 开始
2. 设置 `EnableResume=true`，暂停后继续，观察是否续传
3. 检查日志输出

### 修复 2：下载目录自动创建

**文件：** `MainWindow.xaml.cs`  
**位置：** 约 345-450 行（StartDownload 方法）

**当前问题：**
```csharp
string savePath = configManager.GetDefaultDownloadPath();
string finalPath = Path.Combine(savePath, fileName);
// 未检查目录是否存在
```

**修复代码：**
```csharp
string savePath = configManager.GetDefaultDownloadPath();

// 如果未配置，使用默认目录
if (string.IsNullOrEmpty(savePath))
{
    savePath = AppDomain.CurrentDomain.BaseDirectory;
}

// 确保目录存在
if (!Directory.Exists(savePath))
{
    try
    {
        Directory.CreateDirectory(savePath);
        Debug.WriteLine($"[LOG] 创建下载目录：{savePath}");
    }
    catch (Exception ex)
    {
        Debug.WriteLine($"[LOG] 创建目录失败：{ex.Message}");
        MessageBox.Show($"无法创建下载目录：{savePath}\n{ex.Message}", 
                       "错误", MessageBoxButton.OK, MessageBoxImage.Error);
        return;
    }
}

string finalPath = Path.Combine(savePath, fileName);
```

**测试验证：**
1. 在 config.ini 中设置一个不存在的目录路径
2. 开始下载，观察是否自动创建目录
3. 检查下载是否成功

### 修复 3：最大化按钮图标同步

**文件：** `MainWindow.xaml.cs`  
**位置：** 约 207-212 行（BtnMaximize_Click 方法）

**当前问题：**
```csharp
// 用字符串替换了 PackIcon
btnMaximize.Content = "□";  // 或 "🗗"
```

**修复代码：**
```csharp
private void BtnMaximize_Click(object sender, RoutedEventArgs e)
{
    if (WindowState == WindowState.Normal)
    {
        WindowState = WindowState.Maximized;
        // 修改图标 Kind 而不是替换 Content
        if (btnMaximize.Content is MaterialDesignThemes.Wpf.PackIcon icon)
        {
            icon.Kind = MaterialDesignThemes.Wpf.PackIconKind.WindowRestore;
        }
    }
    else
    {
        WindowState = WindowState.Normal;
        // 修改图标 Kind 而不是替换 Content
        if (btnMaximize.Content is MaterialDesignThemes.Wpf.PackIcon icon)
        {
            icon.Kind = MaterialDesignThemes.Wpf.PackIconKind.CheckboxMultipleBlankOutline;
        }
    }
}
```

**测试验证：**
1. 点击最大化按钮，观察图标是否变化
2. 点击还原按钮，观察图标是否恢复
3. 双击标题栏测试

### 修复 4：按钮背景状态反馈

**文件：** `MainWindow.xaml`  
**位置：** 约 27 行（PrimaryButton 样式定义）

**当前问题：**
```xml
<Border Background="#3B82F6" ...>
    <!-- 硬编码背景色，导致代码中设置的 Background 不生效 -->
</Border>
```

**修复方案 A（推荐）：使用 TemplateBinding**
```xml
<Style x:Key="PrimaryButton" TargetType="Button">
    <Setter Property="Template">
        <Setter.Value>
            <ControlTemplate TargetType="Button">
                <Border x:Name="templateBorder" 
                        Background="{TemplateBinding Background}" 
                        ...>
                    <!-- 其他内容 -->
                </Border>
                <ControlTemplate.Triggers>
                    <Trigger Property="IsMouseOver" Value="True">
                        <Setter TargetName="templateBorder" Property="Background" Value="#1976D2"/>
                    </Trigger>
                </ControlTemplate.Triggers>
            </ControlTemplate>
        </Setter.Value>
    </Setter>
</Style>
```

**修复方案 B（快速）：代码中直接修改模板元素**
```csharp
// 在 StartDownload 中
private async Task StartDownload(string url, string savePath, string connectingMessage, bool preserveUiOnFailure)
{
    // ... 现有代码 ...
    
    btnDownload.Content = "取消";
    
    // 直接修改模板中的 Border
    var border = (Border)btnDownload.Template.FindName("templateBorder", btnDownload);
    if (border != null)
    {
        border.Background = Brushes.Red;
    }
}
```

**测试验证：**
1. 点击下载按钮，观察背景是否变红
2. 点击取消，观察是否恢复蓝色
3. 检查鼠标悬停效果
